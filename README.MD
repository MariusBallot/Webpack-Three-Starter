A simple webpack starter for three.js experiments.<br>

## Requirements
```
NPM
Webpack 3.0 and up
GLSLIFY
```

## Usage
```
git clone https://github.com/MariusBallot/Webpack-Three-Starter.git
npm i
```

### Dev mode
```
npm start
```

### Build
```
npm run build
```
